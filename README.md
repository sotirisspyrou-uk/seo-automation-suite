## 🚀 Quick Start Guide

### Enterprise Installation
```bash
# Clone and setup
git clone https://github.com/sotirisspyrou-uk/seo-automation-suite.git
cd seo-automation-suite

# Enterprise dependencies
pip install -r requirements/enterprise.txt

# Configuration
cp config/enterprise.env.example .env
```

### API Integrations
```python
from seo_suite import TechnicalAuditor, ContentOptimizer

# Initialize with enterprise APIs
auditor = TechnicalAuditor(
    google_api_key=os.getenv('GOOGLE_API_KEY'),
    semrush_api_key=os.getenv('SEMRUSH_API_KEY'),
    screaming_frog_api=os.getenv('SF_API_KEY')
)

# Run comprehensive site audit
audit_results = auditor.run_enterprise_audit(
    domains=['example.com', 'example.co.uk', 'example.de'],
    include_competitors=True,
    generate_executive_summary=True
)
```

### Automated Reporting
```python
from seo_suite import ExecutiveReporter

reporter = ExecutiveReporter()
monthly_report = reporter.generate_board_report(
    timeframe='monthly',
    include_forecasting=True,
    stakeholder_level='c-suite'
)

# Auto-email to stakeholders
reporter.distribute_report(monthly_report, recipients='executives')
```
